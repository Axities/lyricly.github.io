If you want to know how to get BootAnim9 set up, this is the place to be

BootAnim9 does not work using the new boot9strap CFW loading method as of now, this guide only works on A9LH

  1. Download BootAnim9 and extract the contents of the .zip to root (do not replace files)
  2. Rename arm9loaderhax (1).bin on the root of your SD card to arm9loaderhax_si.bin
  3. Move arm9loaderhax.bin on the root of your SD card to the anim folder
  4. Rename arm9loaderhax.bin in the anim folder to arm9payload.bin
  5. Make a 0 folder in the anim folder and extract your boot animation to it

To add another boot animation, make a 1 folder on root; for the next make a 2 folder and so on
